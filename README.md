# CMS-Canteen-management-system-
A complete web-based Canteen Management System built using HTML, CSS, JavaScript, and PHP. This project is designed for college canteens to manage menu items, orders, and staff operations efficiently. It includes separate panels for Customers, Admin, and Staff, making the system organized and easy to use.
